# 🧠 Urdu Mental Health Companion

An AI-powered web application that detects **depression and anxiety** from user text in **Urdu or English** using Natural Language Processing (NLP).

## ✨ Features

- 🌐 Urdu & English support
- 🔄 Automatic Urdu-to-English translation
- 🧠 Depression & anxiety detection
- 📊 Confidence score prediction
- 💙 Personalized coping strategies
- 🚨 Emergency helpline recommendations
- 🔒 Anonymous usage (no login required)

## 🛠️ Tech Stack

- Python
- Flask
- Scikit-learn
- NLP
- Google Translate API

## 📈 Model

- **Dataset:** 35,621 mental health records
- **Accuracy:** 92%
## ⚠️ Disclaimer

This project is for **educational and research purposes only** and is **not a substitute for professional mental health care**.
